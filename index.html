1. Answer 1: The difference between var and let is primarily their scope. Var is a global declaration, although it can be scoped to a function. Let, on the otherhand, is block-scoped, meaning it is tied to the code block where it appears. Also, var can be reassigned and redeclared, whereas, let can only be reassigned
  2. Answer 2: The difference between var and const is that var can be reassigned, redeclared, hoisted, and it global in scope, whereas const cannot be hoisted, reassigned, redeclared, and is block-scoped
  3. Answer 3: The difference between let and const is that the former can be reassigned, but neither can be redeclared
  4. Answer 4: Hoisting is where a var-declared variable actually runs at the top of the code/scope before it appears elsewhere in the code. They can be invoked before they are defined and return "undefined".
